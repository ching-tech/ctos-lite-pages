使用手冊

GitHub Pages 版為展示與說明入口；正式服務需部署 FastAPI、PostgreSQL、LINE Messaging API / Login 與必要的 AI / Google API 設定。

使用者流程

  1. 加入 LINE Bot：@285fjkky。
  2. 直接對話，讓 AI 建立個人化偏好與長期記憶。
  3. 上傳文件或筆記，形成可搜尋的個人 / 公司知識庫。
  4. 透過 Web 後台管理人設、知識、記憶與媒體任務。

核心功能

  • Claude 多輪對話、網路搜尋、圖片理解。
  • 長期記憶、時效記憶與行程提醒狀態列。
  • 語音轉文字、文字轉語音、媒體下載與轉錄。
  • Google Drive 同步，保存知識、記憶與媒體資料。

企業管理

  • 管理員後台可查看用戶、群組、AI 日誌與系統監控。
  • 公司知識 RAG 可匯入 PDF、Word、Excel、CSV、Markdown。
  • 訂閱 / 試用狀態可從後台延長、停用或設定 unlimited。
  • 輸入驗證與 Prompt 注入偵測降低濫用風險。

部署摘要

  • 後端：Python 3.11+、FastAPI、uv、PostgreSQL 16。
  • LINE：Messaging API Channel、LINE Login Channel。
  • 啟動：uv run uvicorn ctos_lite.main:app --host 0.0.0.0 --port 8090
  • Webhook：/api/bot/line/webhook
